# Seibooru

A tag-based image hosting platform, made as a graduation project for the [GOMYCODE](https://gomycode.com/) Fullstack Web Development Bootcamp.

## Running

- `yarn`
- Set up `.env` file using `.env.example` as a template.
- `yarn dev`
